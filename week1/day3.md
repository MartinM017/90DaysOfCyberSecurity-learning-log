#Day 3 --- Networking Functions

## Learning Objective
- learning the networking functions as part of the infrastructure
- study some examples of functions and devices

## Key Concept
- Content Delivery Network (CDN) --- distributing caching server geographically, duplicate the data, speed up the process for users to get data from local server at different location
- Virtual Private Network (VPN) --- it is a secure private data transit tunnel, which encrypt communication on an insecure median
- Quality of Service (QoS) --- function decide the priority of different traffic in network, it classifies traffics and set different priority on different traffic
