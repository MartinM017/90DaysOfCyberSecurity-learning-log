# IP address and Subnetting
## Key Concept
- IP address --- a unique address of each different device
- Subnet mask --- used by the local device to determine the specific subnet using, cannot accross network
- Loopback address --- an address of user itself, used to comfirm local ip address is working
- Reserved address --- set aside for future use or testing, all "class E" address
- Virtual IP Address (VIP) --- not associated with the physical network adapter
- DHCP (Dynamic Host Configuration Protocol) --- it provides automatic address and IP configuration for almost all devices
- APIPA (Automatic Private IP Address) --- a link-local address, can only communicate with other local devices
